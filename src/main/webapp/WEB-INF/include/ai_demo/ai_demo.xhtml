Add a Groq root-CA certificate to the App server truststore, before trying this!

Get certificate via Groq API:

keytool -printcert -rfc -sslserver api.groq.com:443 > c:\temp\groq.pem

- then split it into 3 .pem files (BEGIN-END) -then add only 2. and 3. You can get App server truststore via: System.getProperty("javax.net.ssl.trustStore")
